# TEUM SVG Logo Pack

This package contains clean SVG logo files based on the selected TEUM open-gap / open-door concept.

## Files

- `teum-symbol.svg` : symbol only
- `teum-wordmark.svg` : TEUM wordmark only
- `teum-logo-horizontal.svg` : primary horizontal logo with tagline
- `teum-logo-stacked.svg` : stacked logo with tagline
- `teum-app-icon.svg` : 1024x1024 app icon source

## Recommended Use

- Web / README / Landing page: `teum-logo-horizontal.svg`
- Splash screen: `teum-logo-stacked.svg`
- App icon source: `teum-app-icon.svg`
- Favicon / small mark: `teum-symbol.svg`

## Colors

- Deep Navy: `#102334`
- Teum Green: `#83BFA7`
- Warm White: `#FAFAF8`
- Soft Gray: `#E8ECEF`

## Note

This is a clean vector reconstruction of the selected AI-generated concept.
For final production, open these SVG files in Figma or Illustrator and fine-tune spacing, radius, and typography.
