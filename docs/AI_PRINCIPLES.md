# AI_PRINCIPLES

## Mission

AI reduces friction, not user control.

## AI Should

-   Estimate task duration
-   Suggest the best next task
-   Detect free time
-   Summarize daily progress
-   Break large tasks into smaller actions
-   Suggest realistic schedules

## AI Should Never

-   Automatically delete tasks
-   Schedule without approval
-   Shame users
-   Optimize only for quantity

## Golden Rule

AI recommends. Humans decide.
