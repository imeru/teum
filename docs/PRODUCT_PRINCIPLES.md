# PRODUCT_PRINCIPLES.md

> Every feature should help people use their time more intentionally.

# Core Principle

Time comes before tasks.

Users don't ask:

> I have to do these tasks.

They ask:

> I have 15 minutes. What's the best use of it?

# Product Principles

1.  Reduce cognitive load.
2.  Today comes first.
3.  Break work into small wins.
4.  Every task has an estimated duration.
5.  Recommend work based on context.
6.  Build momentum through consistency.
7.  Calm productivity over busy productivity.
8.  AI assists, users decide.
9.  Notifications must add value.
10. Simplicity beats feature count.

# UX Principles

-   Quiet
-   Minimal
-   Fast
-   Intentional

Users should always know the answer to:

> What should I do next?

# Success Metrics

Measure:

-   Focused time
-   Time slots completed
-   Consistency
-   Meaningful progress

Not simply the number of completed tasks.

# Long-term Vision

Help people make better decisions by making better use of the moments
they already have.
