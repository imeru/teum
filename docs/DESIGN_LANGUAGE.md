# DESIGN_LANGUAGE

## Design Philosophy

TEUM should feel calm, quiet, and intentional.

### Keywords

-   Minimal
-   Spacious
-   Focused
-   Human
-   Fast

## Visual Rules

-   White space is a feature.
-   One primary action per screen.
-   Avoid visual noise.
-   Typography over decoration.

## Color

-   Neutral first.
-   Color communicates meaning, never decoration.
-   Green: progress
-   Blue: focus
-   Amber: attention
-   Red: overdue only

## Interaction

-   One-tap capture.
-   One-tap start focus.
-   Animations should communicate state.
-   Every interaction should feel instant.

## Information Hierarchy

1.  Today
2.  Current Time Slot
3.  Next Task
4.  Calendar
5.  Future
